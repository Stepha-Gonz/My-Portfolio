DESCRIPTION

PROJECT OVERVIEW

This project involved creating a comprehensive Power BI dashboard using data extracted from Excel sheets from a previous analysis project. The main goal was to enhance data visualization by using Power Query to transform and model the data. The dashboard provides detailed insights into key metrics such as the most frequent call patterns, outliers, average performance per team, and the status of pending and completed audits. The dashboard was designed to offer an interactive experience for better understanding and analysis of the data.

The project demonstrates the use of Power BI Service for publishing and sharing the dashboard, showcasing advanced data visualization techniques. It is a continuation of the earlier Excel-based analysis, aimed at providing a more intuitive and dynamic way to explore the data. For confidentiality reasons, the data presented are generalized and created for demonstration purposes only, with no real campaign data used.
